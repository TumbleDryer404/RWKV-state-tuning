## 1. RWKV State Tuning 简介

RWKV 是一种基于 RNN 的语言模型架构，具有线性时间复杂度和低显存占用的优点，适合长上下文任务与高效部署。在 RWKV 体系中，State Tuning（状态微调）是一种高效的参数微调方法，其核心思想是：仅微调模型的初始隐藏状态（state），而不修改原始的模型权重。由于 RWKV 是纯 RNN 结构，模型的行为在很大程度上由其内部状态决定。因此，微调 state 是最彻底的 Prompt Tuning ，训练参数极少且显存占用低，适合单卡训练。

训练完成后将会得到 `.pth` 格式的 **state 文件**，它可以与原始 RWKV 模型组合使用，或者合并为完整模型，以及改变模型的行为风格、知识偏向或任务能力
